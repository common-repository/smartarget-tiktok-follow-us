== Description ==

Turn visitors into TikTok Followers
Do you have a TikTok account? Our app allows you to tell your users about your TikTok account so they can subscribe and watch your TikTok content .
TikTok is now one the most popular social networks, so it is important to use it.
App will show a non intrusive but yet very visible message to your users and recommend them to follow you.
Smartarget TikTok  plugin works also with WooCommerce, Elementor and Oxygen Builder.

== Installation ==

1. Unpack plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings page
4. Configure settings and press Save
